# Quickstart Guide

This guide will help you get up and running with Solana Agent Kit quickly.

## Installation

```bash
npm install solana-agent-kit
```

## Basic Setup

First, import and initialize the Solana Agent:

```typescript
import { SolanaAgent } from 'solana-agent-kit';

const agent = new SolanaAgent({
    network: 'devnet', // or 'mainnet-beta'
    // Add your configuration options here
});
```

## Common Operations

### Checking Account Balance

```typescript
const balance = await agent.getBalance('ADDRESS');
console.log(`Balance: ${balance} SOL`);
```

### Creating a New Token

```typescript
const tokenMint = await agent.createToken({
    decimals: 9,
    initialSupply: 1000000000
});
console.log(`Created token: ${tokenMint}`);
```

### Working with NFTs

```typescript
const nft = await agent.createNFT({
    name: "My NFT",
    symbol: "MNFT",
    uri: "https://your-metadata-uri.json"
});
```
